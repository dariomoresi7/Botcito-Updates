Facing an "Incompatible Operating System" error?
This installer requires Windows 10 Build 19041 (Version 2004) or higher. If your deployment gets blocked, please update your operating system:
  1. Open your Start Menu and go to Settings (⚙️).
  2. Navigate to Update & Security > Windows Update.
  3. Click "Check for updates", install all pending features, and restart your PC.
  4. Alternatively, download the official Windows 10 Update Assistant tool directly from Microsoft's website to force the upgrade.
System Requirements: Compatible exclusively with Windows 11 or Windows 10 (Version 2004 / Build 19041 or higher). Administrative permissions are automatically managed via the application manifest. All subsequent updates are synchronized invisibly on application startup.
